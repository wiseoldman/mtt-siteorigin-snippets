# MTT SiteOrigin Snippets

Snippet pack developed at Morning Train for SiteOrigin pagebuilder snippets.

## Snippets

### Create widget

```
widget
```

### Widget text

```
wtext
```

### Widget image

```
wimage
```

### Widget link

```
wlink
```

### Widget color

```
wcolor
```

### Widget repeater

```
wrepeater
```
