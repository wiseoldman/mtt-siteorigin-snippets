# MTT Snippet Pack

## Features

Snippet pack developed at Morning Train for WP and JS (SiteOrigin pagebuilder snippets + general snippets)

## Snippets

### SiteOrigin Pagebuilder

#### Create widget
```
widget
```

#### Widget text
```
wtext
```

#### Widget image
```
wimage
```

#### Widget link
```
wlink
```

#### Widget color
```
wcolor
```
